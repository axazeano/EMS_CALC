GUI for work with EMS API, based on Tkinter and TTK

**Required:**
* Python 3.2 or later
* For Linux based OS need to install ```python-tkinter``` package

**Launch:**
* Run GUI.py

*In case of launch on Windows OS, be sure that correct environment variables are set*
